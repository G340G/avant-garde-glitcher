# Avant-Garde Datamosh & Glitch Studio

A highly customizable command-line tool built in Python to corrupt, datamosh, and add an experimental aesthetic to your videos. Using calculated frame differentials and pixel bleeding algorithms, it safely replicates high-fidelity compression corruption without ruining video container playability.

## 🚀 Features

* **P-Frame Datamoshing:** Mimics compression pixel drag artifacts by bleeding color and motion data from previous frames into current ones.
* **Chromatic Aberration:** Separates and shifts RGB color channels to provide a retro/analog hardware distortion.
* **Scanline/Slice Corruption:** Randomly tears and shifts horizontal slices of video data for heavy glitch stylization.
* **Granular Control:** Dynamically scale the intensity ("entity") of each effect via arguments.
